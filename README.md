# Bowl Builder

Kleine statische Website, die per Klick eine zufällige Bowl aus fünf Kategorien erzeugt.

## Projekt öffnen

1. Repository lokal herunterladen oder klonen.
2. Die Datei `index.html` im Browser öffnen (Doppelklick genügt).

Optional: Falls du lieber mit einem lokalen Server arbeitest, kannst du z. B. im Projektordner ausführen:

```bash
python3 -m http.server 8000
```

Dann im Browser öffnen: `http://localhost:8000`

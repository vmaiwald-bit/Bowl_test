# Bowl Builder

Kleine statische Website (inkl. einfacher PWA-Basis), die per Klick eine zufällige Bowl aus fünf Kategorien erzeugt.

## Projekt öffnen

1. Repository lokal herunterladen oder klonen.
2. Datei `index.html` direkt im Browser öffnen **oder** einen lokalen Server starten.

Optionaler lokaler Server:

```bash
python3 -m http.server 8000
